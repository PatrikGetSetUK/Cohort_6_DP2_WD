Basic text formatting in HTML

Text tags

To add a paragraph in your HTML code, you should use the "p" tag. This will display the text "This is a paragraph" in a separate block on your webpage.

As default the paragraph is created on the new line.

The text tags will automatically strip all extra spaces at the beginning, between, and at the end of the words/sentence. To adjust the text to our needs we will use CSS language later on.

To add some extra spacing between HTML elements you can use "br" tag



As you can see some extra space between this a above paragraph was add.

What about in-line text? Simply use "span" tag. It doesn't matter how many span you will place in the code bellow each other, they will always appear in-line nex to each other.

Span can be used within a paragraph. Like here. Remember that to use "p" tag inside the span is not convenient.

What about to make some text in bold? Or italic? As you can see bold and italic elements are in-line elements as well as span.

Of corse you can use bold text or italic text inside the paragraph. You can use bold and italic text within the "span" tag too.

The HTML "del" element defines text that has been deleted from a document. Browsers will usually strike a line through deleted text. The HTML "ins" element defines a text that has been inserted into a document. Browsers will usually underline inserted text.

Use horizontal line, "hr" tag to visually divide web page.

What about visual disability and the text in HTML?

Screen Reader apps for the people with visual disability will often require "strong" and "em" tags in order for the voice to stress those words properly.

The "strong" tag is used to define text with strong importance. The content inside is typically displayed in bold.

The "em" tag is used to define emphasized text. The content inside is typically displayed in italic.
